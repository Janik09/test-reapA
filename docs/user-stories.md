# User Stories – RestaurantApp (MVP)

M = MustHave
S = Sould Have
C = Could Have

## Reservierung & Bestellung
- Als User möchte ich einen Tisch auswählen und reservieren können, damit ich sicher einen Platz habe. (M)
- Als User möchte ich Beginn und Dauer meiner Reservierung auswählen können, damit das Restaurant meine Ankunft einplanen kann. (M)
- Als User möchte ich die aktuelle Speisekarte anzeigen können, damit ich weiß, welche Gerichte angeboten werden. (M)
- Als User möchte ich einzelne Gerichte vorab auswählen und bestellen können, damit sie pünktlich bei meiner Ankunft bereitstehen. (M)

## Zahlung & Anmeldung
- Als User möchte ich ein Konto erstellen können, damit meine Reservierungen gespeichert werden. (M)
- Als User möchte ich mich ein- und ausloggen können, damit ich meine persönlichen Daten sicher verwalten kann. (M)
- Als User möchte ich meine Rechnung online begleichen können, damit ich das Restaurant schneller verlassen kann. (C)

## Benachrichtigung & Verwaltung
- Als User möchte ich nach einer erfolgreichen Reservierung eine Benachrichtigung erhalten, damit ich sicher sein kann, dass der Tisch für mich reserviert wurde. (S)
- Als User möchte ich eine bestehende Reservierung stornieren können, damit der Tisch freigegeben wird, wenn ich doch nicht kommen kann. (S)

## Restaurant-Perspektive
- Als Koch möchte ich Gerichte als verfügbar oder nicht verfügbar markieren können, damit keine Bestellungen für nicht verfügbare Gerichte eingehen. (M)
- Als Koch möchte ich eine Liste aller aktiven Bestellungen einsehen können, damit ich effizient planen kann. (M)
- Als Koch möchte ich alle zukünftigen Reservierungen sehen können, damit ich die Arbeitsmenge besser einschätzen kann. (S)
- Als Koch möchte ich Bestellungen als in Bearbeitung markieren können, damit der Service den Küchenstatus kennt. (S)

- Als Kellner möchte ich sehen können, welche reservierten Tische mir zugeordnet sind, damit ich meine Aufgaben planen kann. (S)

- Als Restaurantbesitzer möchte ich eine Übersicht aller Reservierungen sehen können, damit ich die Auslastung des Restaurants planen kann. (M)
- Als Restaurantbesitzer möchte ich Gerichte hinzufügen, bearbeiten oder deaktivieren können, damit die Speisekarte aktuell bleibt. (M)
- Als Restaurantbesitzer möchte ich tägliche oder wöchentliche Umsätze einsehen können, damit ich wirtschaftliche Entscheidungen treffen kann. (S)
- Als Restaurantbesitzer möchte ich das Feedback der Gäste sehen können, damit ich den Service verbessern kann. (C)
